## show-image

### 介绍

用于实现 KPlayer 推流过程中在指定位置显示图片的插件

### 参数

### 参数列表

| 参数     | 默认值      | 取值范围 | 说明                                                               |
| -------- | ----------- | -------- | ------------------------------------------------------------------ |
| fontname | example.png | 无       | 图片文件路径                                                       |
| x        | 0           | 整型     | x 轴坐标值值。以屏幕左上角为原点，最大可显示范围为当前设置的分辨率 |
| y        | 0           | 整型     | y 轴坐标值值。以屏幕左上角为原点，最大可显示范围为当前设置的分辨率 |
